__define size as a factor (categorical variable)__
```r
size<-c("S", "M","L","XL")
size_factor<-factor(size)
size_factor
```
```r
[1] S  M  L  XL
Levels: L M S XL
```
L M S XL 알페벳 순서로 배열된다 
해결방법


```r
size_factor2 <- factor(size, 
                       levels = c("S", "M","L","XL")) 
size_factor2
```
```r
[1] S  M  L  XL
Levels: S M L XL
```
```r
size_factor3 <- factor(size, ordered = TRUE, 
                       levels = c("S", "M","L","XL")) 
size_factor3
```
```r
[1] S  M  L  XL
Levels: S < M < L < XL
```
