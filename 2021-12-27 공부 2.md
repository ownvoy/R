### lattice  

__직교 형태의 그래픽 툴__

- xyplot: 산점도
- bwplot: 상자그림
- dotplot
- levelplot
- stripplot: 점을 함께 표시한 상자그림
- splom: 산점도 매트릭스
- contourplot: 등고선
<br>

#### basic xyplot  
```r
xyplot(NOx ~ E , data = ethanol, main = "Single Panel by xyplot")
```

#### multi panel graph  
```r
xyplot(NOx ~ E | C , data = ethanol, main = "NOx ~ E | C ")
```
![조건부 산점도](https://user-images.githubusercontent.com/96481582/147444385-75ef6989-682e-49b5-81a9-27b4c7cc86a6.png)

#### multi panel graph for subset
```r
xyplot(NOx ~ E | C , data = ethanol, subset = C > 8,
       main = "NOx ~ E | C , data = ethanol, subset = C > 8")
```
